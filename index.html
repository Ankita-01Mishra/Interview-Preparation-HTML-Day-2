1. Difference between HTML tags and elements.

Ans:- In html, tags consist of opening brackets and closing bracket , whereas, elements consists of a starting tag, content and an end tag.Whatever written within a HTML tag are HTML elements. HTML tags are used to hold the HTML element, on the other hand, HTML element holds the content.

EXAMPLE:

2. Explain br tags

Ans:- The br tag insert a single line break. The br tag is useful for writing addresses and poems. The br tag is an empty tag which means that it has no end tag.

3. Explain how many different types of lists we have in html

Ans:- There are 3 types of list in html:
a) Ordered list
b) Unordered list
c) Descriptive list
a) Ordered list -> The ordered lists have an order which is either numerical or alphabetical . Here we use li tag inside of ol tag.

  1. Apple
  2. Banana
  3. Mango

b) Unordered list -> The unordered list is used to create a list of related items,in bulleted form or unordered format using type="circle" type="disc" type="square" type="none". type="disc" is default here.

  • Apple
  • Banana
  • Mango

c) Descriptive list -> The unordered list is a type of list where each item has a description. It is also known as a definition list.The dl tag is used to create description list

Coffee
Black hot drink
Milk
White cold drink

4. What are semantic tags?

Ans:- Semantic tags are the types of tags that tells about itself to browser and developer as well. It helps in Search Engine Optimization. Example: form , table , article , header , footer
